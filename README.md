## SDKs & Libraries

- [velixar-python](https://github.com/VelixarAi/velixar-python) — Python SDK. `pip install velixar`
- [velixar-js](https://github.com/VelixarAi/velixar-js) — JavaScript/TypeScript SDK. `npm install velixar`

## Developer Tools

- [velixar-cli](https://github.com/VelixarAi/velixar-cli) — Command-line interface for managing memories.
- [velixar-tui](https://github.com/VelixarAi/velixar-tui) — Terminal UI for browsing and searching memories.
- [velixar-vscode](https://github.com/VelixarAi/velixar-vscode) — VS Code extension.
- [velixar-mcp-server](https://github.com/VelixarAi/velixar-mcp-server) — MCP server for Claude, Cursor, and VS Code Copilot.

## Integrations

- **LangChain** — [VelixarMemory class](https://github.com/VelixarAi/velixar-examples/tree/main/python/langchain-memory) — Drop-in memory backend for any chain.
- **CrewAI** — [Shared agent memory](https://github.com/VelixarAi/velixar-examples/tree/main/python/crewai-agents) — Multi-agent systems with shared recall.
- **OpenAI Assistants** — [Function calling](https://github.com/VelixarAi/velixar-examples/tree/main/python/openai-assistants) — remember/recall tools for Assistants API.
- **Vercel AI SDK** — [Streaming + memory](https://github.com/VelixarAi/velixar-examples/tree/main/js/vercel-ai-sdk) — Streaming chat with persistent context.
- **FastAPI** — [Per-user chatbot](https://github.com/VelixarAi/velixar-examples/tree/main/python/fastapi-chatbot) — REST API with per-user memory.
- **Next.js** — [API route](https://github.com/VelixarAi/velixar-examples/tree/main/js/nextjs-app) — Full-stack app with AI memory.
- **Discord** — [Memory bot](https://github.com/VelixarAi/velixar-examples/tree/main/js/discord-bot) — Bot that remembers users.
